Question Extraction API

AI-Powered Educational Content Extraction using Azure Document Intelligence & OpenAI GPT-4

About This Solution

This API automatically extracts educational questions from PDF textbooks using advanced AI technology. It combines Azure Document Intelligence for precise text extraction with OpenAI GPT-4 for intelligent question identification and categorization.

Perfect for educators, educational institutions, and content creators who need to digitize and organize questions from Arabic and English educational materials while preserving mathematical symbols, chemical formulas, and scientific notation.

API Endpoints

POST /extract

Extract questions from PDF files using AI

curl -X POST "{{ request.url.scheme }}://{{ request.url.netloc }}/extract" \ -F "file=@textbook.pdf" \ -F "subject_name=الرياضيات" \ -F "class_name=الصف الثاني عشر" \ -F "uploaded_by=admin"
GET /questions

List all questions with optional filters

curl "{{ request.url.scheme }}://{{ request.url.netloc }}/questions?subject_name=الرياضيات&limit=10"
POST /users

Create new user account

curl -X POST "{{ request.url.scheme }}://{{ request.url.netloc }}/users" \ -H "Content-Type: application/json" \ -d '{"username":"teacher1","password":"pass123","display_name":"أستاذ أحمد"}'
POST /login

Authenticate user

curl -X POST "{{ request.url.scheme }}://{{ request.url.netloc }}/login" \ -H "Content-Type: application/json" \ -d '{"username":"teacher1","password":"pass123"}'
