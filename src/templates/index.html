About This Solution

This API automatically extracts educational questions from PDF textbooks using advanced AI technology. It combines Azure Document Intelligence for precise text extraction with OpenAI GPT-4o for intelligent question identification and categorization.

Perfect for educators, educational institutions, and content creators who need to digitize and organize questions from Arabic and English educational materials while preserving mathematical symbols, chemical formulas, and scientific notation.
